Steps to locally run the project:

1) Checkout the code
2) Download all gradle dependencies
3) Run ShoppingApplication.java (com/bullish/shopping/ShoppingApplication.java)

All the endpoints will be available to use via Swagger

4) Open web browser and go to the link : http://localhost:8082/swagger-ui.html#/

This application runs with in memory H2 database and this DB will be available at : localhost:8082/h2-console/
    Username : admin
    Password: admin


Test Cases :

This project is equipped with springboot testing environment and all the test cases can be run by
running the class ShoppingApplicationTests.java (com/bullish/shopping/ShoppingApplicationTests.java)
